# Анализ факторов, влияющих на количество навыков

**Цель исследования**
Определить, влияют ли опыт работы и количество сертификатов на количество навыков.

**Данные**
Датасет был взят из платформы Kaggle. Ссылка на источник: https://www.kaggle.com/datasets/nalisha/job-salary-prediction-dataset/data

**Методы**
Линейная регрессия, EDA (Разведочный анализ данных), Pandas, Scikit-learn

**Достигнутые результаты**
- Анализ влияния опыта работы:
MSE (средний квадрат ошибок) = 30.0. RMSE = 5.47 при размахе целевой переменной от 1 до 19, что составляет 30 % от выборки.
Это говорит о том, что опыт работы не имеет значительного влияния на количество навыков.
R^2 (доля дисперсии) = -8; R^2 < 0. Это означает, что модель не объясняет данные в принципе, что тоже говорит об отсутствии взаимосвязи между опытом работы и количеством навыков.

- Анализ влияния количества сертификатов:
Путём группировки количества навыков('skills_count') по количеству сертификатов ('certifications') выяснилось, что среднее количество навыков при разном количестве сертификатов (0-5) приблизительно равно 10.
Это говорит о том, что взаимосвязи количества сертификатов с количеством навыков тоже нет.

**Выводы**
В данном датасете ни опыт работы, ни количество сертификатов не являются значимыми признаками количества навыков сотрудников. 
Вероятнее всего, в датасете не были учтены ключевые факторы влияния, либо датасет был сгенерирован случайно.

**Примечание**
Несмотря на выполненное импортирование библиотеки matplotlib для визуализации данных, мною было принято решение не использовать её в анализе данных в связи с отсутствием взаимосвязей факторов.

**Используемые библиотеки**
Pandas, Scikit-learn
